ShadowBuster 🕶️

That black redaction bar isn't redaction — it's a sticker.

Drop a PDF, Excel, Word, PowerPoint, or screenshot. ShadowBuster surfaces what was hidden but never actually removed — text under black boxes, very-hidden sheets, deleted tracked changes, the uncropped original of a cropped image.

🔒 100% in your browser. The file never leaves this page — no upload, no server, no network.

Why this exists: "redacting" by drawing a black box, hiding a sheet, or cropping an image removes nothing — the data is still in the file you sent. ShadowBuster shows you, before you hit send.

Same engine in your terminal — batch-scan a whole folder: npx shadowbuster ./documents. GitHub · MIT

Use it on your own files. ShadowBuster only reveals what a document already carries to anyone who opens it — it's an awareness tool, not an exploit. To truly redact: flatten the PDF / export to image, delete hidden sheets, accept tracked changes, and re-export cropped images. Everything here runs locally; nothing you drop is uploaded or logged.
